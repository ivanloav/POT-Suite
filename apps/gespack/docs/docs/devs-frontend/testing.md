# Testing y buenas prácticas

## Testing
- Usa Jest y React Testing Library para pruebas unitarias y de integración.
- Los tests se ubican junto a los componentes o en carpetas `__tests__`.
- Ejecuta los tests con:

```bash
pnpm test
```

## Buenas prácticas
- Mantén los componentes pequeños y reutilizables.
- Usa hooks para lógica compartida.
- Documenta cada nuevo componente y hook.
- Añade traducciones para nuevos textos.
- Haz revisiones de código antes de mergear.
