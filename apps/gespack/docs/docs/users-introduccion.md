# Guía para usuarios de GesPack

Bienvenido a GesPack, la plataforma para gestionar pedidos, clientes y productos de forma sencilla y segura.

## ¿Qué puedes hacer con GesPack?
- Crear y gestionar pedidos
- Consultar y editar clientes
- Gestionar productos y líneas de pedido
- Configurar pagos y cuotas
- Añadir observaciones internas y externas
- Exportar estadísticas y datos

## Primeros pasos
1. Inicia sesión con tus credenciales.
2. Accede al menú principal para ver las secciones disponibles.
3. Para crear un pedido, haz clic en "Crear pedido" y completa los datos requeridos.

## Secciones principales
- **Pedidos**: Listado, búsqueda y edición de pedidos.
- **Clientes**: Consulta y edición de datos de clientes.
- **Productos**: Gestión de productos y líneas de pedido.
- **Pagos**: Configuración de tipo de pago, cuotas y promociones.
- **Observaciones**: Añade notas internas (solo personal) y externas (visibles para el cliente).

## Preguntas frecuentes
- ¿Cómo busco un pedido? Usa los filtros en la lista de pedidos.
- ¿Cómo añado observaciones? En el formulario de pedido, sección "Observaciones".
- ¿Cómo exporto datos? Usa el botón "Exportar estadísticas" en la sección de pedidos.

## Ayuda interactiva
En el futuro, GesPack contará con un chatbot integrado (OpenAI) para resolver dudas y guiarte en la navegación y uso de la plataforma.
