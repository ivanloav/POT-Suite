# Cómo editar un pedido en GesPack

Esta guía te explica cómo modificar un pedido existente desde la interfaz de GesPack.

## 1. Acceso a la lista de pedidos
- Ve al menú principal y selecciona "Pedidos".
- Busca el pedido que deseas editar usando los filtros disponibles (ID, cliente, estado, etc).

## 2. Selección y edición
- Haz clic en el pedido que quieres modificar.
- Se abrirá el formulario de edición con los datos actuales.

## 3. Modifica los datos necesarios
- Puedes cambiar cualquier campo permitido: cliente, productos, cantidades, tipo de pago, observaciones, etc.
- Si el pedido ya está pagado o facturado, algunos campos pueden estar bloqueados por seguridad.

## 4. Validación y guardado
- Revisa los cambios y asegúrate de que los datos sean correctos.
- Pulsa "Guardar" para aplicar las modificaciones.
- El sistema validará los datos y mostrará mensajes de error si es necesario.
- Al guardar, verás una confirmación y el pedido actualizado en la lista.

## 5. Consejos y ayuda
- Usa los tooltips para ver información adicional sobre cada campo.
- Si tienes dudas, pronto podrás consultar el chatbot integrado.
